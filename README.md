# Radiant Ledger

Live Deployment: [Radiant Ledger on Vercel](https://radiant-ledger-ten.vercel.app)

## Local Development

To run this project locally:

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:5173` in your browser.
